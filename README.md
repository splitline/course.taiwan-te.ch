## build enviroment
```
$ git clone https://github.com/splitline/course.taiwan-te.ch/
$ cd course.taiwan-te.ch
$ virtualenv env
$ . env/bin/activate
$ pip install -r requirements.txt

$ cd taiwan_tech_course
$ python manage.py runserver
```
